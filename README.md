# hexo-tag-link-preview
Embed a link preview on your [Hexo](https://hexo.io/) article.

## Installation

`npm install hexo-tag-link-preview`

or

`yarn add hexo-tag-link-preview`

## Usage
`{% linkPreview url %}`

Example:
```
{% linkPreview https://www.amazon.com/ %}
```

it generates HTML:

